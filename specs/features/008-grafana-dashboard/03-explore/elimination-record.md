## Detail Records

### OPT-C — Custom Python/Streamlit Dashboard

**Eliminated At:** Pre-exploration scope decision, 2026-05-16
**Eliminated By:** Chris

**Description:**
A custom Python dashboard using Streamlit as the UI framework and Plotly/Altair for chart rendering. A prior implementation exists at `../hem_dashboard`: a Streamlit + Plotly dashboard with dual-axis temperature/humidity plotting, room selection, time-range filtering, and Docker Compose deployment. This implementation was vibe-coded with an early LLM and is currently broken.

**Evidence of Elimination:**
- The prior implementation (`../hem_dashboard`) is currently non-functional.
- The codebase has accumulated significant complexity: Pipfile, pyproject.toml, requirements.txt, multiple modules (db_connection, dashboard_utils, constants, data_model), and mixed Plotly/Altair chart types — all generated by LLM without systematic design.
- The operator explicitly confirmed this approach is too complex and time-intensive for the current scope, given the primary value is speed to a working dashboard.
- A clean rebuild would require the same development effort as the original, with no guarantee of avoiding the same complexity accumulation.

**Conditions for Reconsideration:**
This approach may be reconsidered in a future iteration if: (a) the operator has a specific visualization requirement that neither Grafana nor Metabase can satisfy, and (b) sufficient development time is available. The `hem_dashboard` codebase can serve as a reference implementation.
